Завдання 4 - Функція доставки товару

Умова: Оголосити функцію getShippingMessage, яка очікує три параметри: country (країна доставки), price (вартість товару), deliveryFee (вартість доставки). Функція повертає рядок з повідомленням про доставку.

Код програми:

// Оголосити функцію getShippingMessage function getShippingMessage(country, price, deliveryFee) { // Обчислити загальну вартість (товар + доставка) let totalPrice = price + deliveryFee; // Повернути рядок з повідомленням return `Shipping to ${country} will cost ${totalPrice} credits`; } // Тестова функція function testShipping() { console.log("=== Тестування функції getShippingMessage ==="); // Тест 1 let result1 = getShippingMessage("Australia", 120, 50); console.log(result1); alert(result1); // Тест 2 let result2 = getShippingMessage("Germany", 80, 20); console.log(result2); // Тест 3 let result3 = getShippingMessage("Sweden", 100, 20); console.log(result3); console.log("=== Тестування завершено ==="); }

Приклади виклику:

getShippingMessage("Australia", 120, 50) // Результат: "Shipping to Australia will cost 170 credits" getShippingMessage("Germany", 80, 20) // Результат: "Shipping to Germany will cost 100 credits" getShippingMessage("Sweden", 100, 20) // Результат: "Shipping to Sweden will cost 120 credits"
