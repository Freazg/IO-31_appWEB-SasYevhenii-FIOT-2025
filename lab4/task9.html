Завдання 9 - Візуальний елемент «Спливаюча підказка»

Умова завдання:

Надати можливість виведення спливаючої підказки при наведенні миші над текстом Web-сторінки. Передбачити можливість створення нових підказок та їх підключення до обраного фрагменту тексту.

Демонстрація роботи:

JavaScript Мова програмування для веб-розробки - це потужна мова програмування, яка використовується для створення інтерактивних веб-сторінок.

Вона підтримує масиви Структура даних для зберігання елементів , функції Блок коду, який можна викликати багато разів , та об'єкти Колекція властивостей та методів .

За допомогою DOM Document Object Model - API для роботи з HTML можна динамічно змінювати вміст сторінки.

HTML5 П'ята версія мови розмітки HTML та CSS3 Каскадні таблиці стилів версії 3 разом створюють сучасні веб-додатки.

HTML код:

<span class="tooltip-trigger">JavaScript <span class="tooltip">Мова програмування для веб-розробки</span> </span>

CSS код:

/* Стилі для елементів з підказкою */ .tooltip-trigger { color: #667eea; font-weight: bold; cursor: help; border-bottom: 2px dotted #667eea; position: relative; display: inline-block; } /* Стилі для спливаючої підказки */ .tooltip { position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); margin-bottom: 10px; padding: 10px 15px; background: #333; color: white; border-radius: 6px; font-size: 14px; white-space: nowrap; opacity: 0; visibility: hidden; transition: opacity 0.3s, visibility 0.3s; z-index: 1000; } /* Стрілка підказки */ .tooltip::after { content: ''; position: absolute; top: 100%; left: 50%; transform: translateX(-50%); border: 6px solid transparent; border-top-color: #333; } /* Показати підказку при наведенні */ .tooltip-trigger:hover .tooltip { opacity: 1; visibility: visible; }

JavaScript функції для динамічного створення підказок:

// Функція для додавання підказки до елемента function addTooltip(elementId, tooltipText) { let element = document.getElementById(elementId); if (!element) { console.error(`Елемент з ID "${elementId}" не знайдено`); return; } // Додати клас для стилізації element.classList.add('tooltip-trigger'); // Створити елемент підказки let tooltip = document.createElement('span'); tooltip.className = 'tooltip'; tooltip.textContent = tooltipText; // Додати підказку до елемента element.appendChild(tooltip); console.log(`Підказку "${tooltipText}" додано до елемента "${elementId}"`); } // Функція для додавання підказки до тексту за селектором function addTooltipToSelector(selector, tooltipText) { let elements = document.querySelectorAll(selector); elements.forEach(function(element) { element.classList.add('tooltip-trigger'); let tooltip = document.createElement('span'); tooltip.className = 'tooltip'; tooltip.textContent = tooltipText; element.appendChild(tooltip); }); console.log(`Підказки додано до ${elements.length} елементів`); } // Приклади використання: // addTooltip('myElement', 'Це моя підказка'); // addTooltipToSelector('.term', 'Технічний термін');

Переваги реалізації:

  • Чистий CSS без JavaScript для базової функціональності
  • Плавна анімація появи/зникнення
  • Адаптивне позиціонування
  • Стрілка вказує на елемент
  • Можливість динамічного додавання підказок через JavaScript
  • Легко налаштовувані стилі

Інструкція: Наведіть курсор миші на виділені слова у жовтому блоці "Демонстрація роботи", щоб побачити спливаючі підказки.
