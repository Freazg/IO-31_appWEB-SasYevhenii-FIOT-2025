Завдання 8 - Login Form

Умова: Обробка відправлення форми за подією submit. Сторінка не повинна перезавантажуватися. Якщо є незаповнені поля - вивести alert. Зібрати значення полів в об'єкт і вивести в консоль, очистити форму.

Пояснення:

Використовується подія submit форми, метод preventDefault() для запобігання перезавантаженню, властивість elements для доступу до полів, trim() для очищення пробілів, метод reset() для очищення форми.
