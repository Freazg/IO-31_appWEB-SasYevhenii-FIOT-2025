# Трекер Звичок (Habit Tracker)

> Веб-застосунок для формування та відстеження корисних звичок

## Опис проєкту

Трекер звичок - це інтуїтивний веб-застосунок, який допомагає користувачам формувати корисні звички через систематичне відстеження прогресу та візуалізацію досягнень.

### Основні можливості

- Створення та управління звичками
- Відстеження щоденного виконання
- Візуалізація прогресу (календар, графіки)
- Система мотивації та досягнень
- Адаптивний дизайн для всіх пристроїв

## Реалізовані елементи

### HTML-теги та структура:

- Заголовки - h1, h2, h3, h4 різних рівнів
- Зображення - img з атрибутами alt, src
- Таблиці - table, thead, tbody, tr, th, td
- Списки:
  - Нумеровані списки ol, li
  - Маркіровані списки ul, li
  - Вкладені списки
- Форми - form з різними типами input:
  - type="text" - текстові поля
  - type="email" - email
  - type="date" - дата
  - type="radio" - радіо-кнопки
  - type="checkbox" - чекбокси
  - textarea - багаторядкове текстове поле
  - select - випадаючий список
  - button - кнопка відправки
- Посилання - a внутрішні та зовнішні
- Семантичні теги - header, nav, section, footer

## Предметна галузь

Застосунок призначений для:

1. **Управління звичками** - створення, редагування, видалення

3. **Візуалізація даних** - календарі, графіки, статистика
4. **Мотивація** - досягнення, нагадування
