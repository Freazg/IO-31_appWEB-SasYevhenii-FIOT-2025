Завдання 2.2 - Перевірка віку (some)

Умова: Дано масив об'єктів. Перевірте, чи є хоча б один об'єкт з віком менше 20 років. Очікуваний результат: true.

Код програми:

const people = [ { name: 'John', age: 27 }, { name: 'Jane', age: 31 }, { name: 'Bob', age: 19 }, ]; function checkAge() { console.log("=== ПЕРЕВІРКА ВІКУ ==="); console.log("Масив людей:"); console.log(people); console.log(""); // Перевірка чи є хоча б один об'єкт з віком менше 20 років const hasYoungPerson = people.some(person => person.age < 20); console.log("Результат перевірки:"); console.log(`Чи є хоча б одна особа молодше 20 років? ${hasYoungPerson}`); console.log(""); // Виведення деталей console.log("Деталі:"); people.forEach(person => { const status = person.age < 20 ? "молодше 20" : "старше 20"; console.log(`${person.name}: ${person.age} років (${status})`); }); // Пошук усіх молодих людей const youngPeople = people.filter(person => person.age < 20); console.log(""); console.log(`Знайдено ${youngPeople.length} осіб молодше 20 років:`); youngPeople.forEach(person => { console.log(`- ${person.name}, ${person.age} років`); }); alert(`Результат: ${hasYoungPerson}\n\nЧи є особа молодше 20 років? ${hasYoungPerson ? "Так" : "Ні"}\n\nДив. Console для деталей`); }

Пояснення:

  • some() - перевіряє чи хоча б один елемент відповідає умові
  • person => person.age < 20 - стрілочна функція для перевірки віку
  • Повертає true - якщо хоча б один елемент проходить перевірку
  • Повертає false - якщо жоден елемент не проходить перевірку
  • filter() - додатково використовується для пошуку всіх молодих людей
