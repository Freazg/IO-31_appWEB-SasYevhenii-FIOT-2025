Завдання 1.4 - Concerts (Стрілочні функції)

Умова: З об'єкту concerts отримати масив, який буде містити лише імена міст. З масиву потрібно прибрати міста, в яких концерт уже пройшов і відсортувати їх у хронологічному порядку. Очікуваний результат: ["Одеса", "Умань", "Харків"]. Застосувати стрілочні функції.

Код програми:

const concerts = { Київ: new Date("2020-04-01"), Умань: new Date("2025-07-02"), Вінниця: new Date("2020-04-21"), Одеса: new Date("2025-03-15"), Хмельницький: new Date("2020-04-18"), Харків: new Date("2025-07-10"), }; function processConcerts() { console.log("=== ОБРОБКА КОНЦЕРТІВ ==="); console.log("Початковий об'єкт:"); console.log(concerts); console.log(""); // Поточна дата const today = new Date(); console.log(`Поточна дата: ${today.toLocaleDateString()}`); console.log(""); // Отримати масив міст з майбутніми концертами // Використовуємо стрілочні функції const futureConcerts = Object.keys(concerts) .filter(city => concerts[city] > today) .sort((a, b) => concerts[a] - concerts[b]); console.log("Міста з майбутніми концертами (відсортовані):"); console.log(futureConcerts); console.log(""); console.log("Деталі майбутніх концертів:"); futureConcerts.forEach(city => { console.log(`${city}: ${concerts[city].toLocaleDateString()}`); }); alert(`Майбутні концерти:\n\n${futureConcerts.join(", ")}`); }

Пояснення:

  • Object.keys(concerts) - отримуємо масив ключів (назв міст)
  • filter(city => concerts[city] > today) - фільтруємо тільки майбутні концерти
  • sort((a, b) => concerts[a] - concerts[b]) - сортуємо за датою у хронологічному порядку
  • Всі операції використовують стрілочні функції
