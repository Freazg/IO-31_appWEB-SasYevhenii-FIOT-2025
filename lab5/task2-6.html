Завдання 2.6 - Сортування за віком (sort)

Умова: Відсортуйте масив об'єктів за віком у порядку зростання. Очікуваний результат: [{name: "Bob", age: 19}, {name: "John", age: 27}, {name: "Jane", age: 31}].

Код програми:

const users = [ { name: 'John', age: 27 }, { name: 'Jane', age: 31 }, { name: 'Bob', age: 19 }, ]; function sortByAge() { console.log("=== СОРТУВАННЯ ЗА ВІКОМ ==="); console.log("Початковий масив:"); console.log(users); console.log(""); console.log("Користувачі до сортування:"); users.forEach(user => { console.log(`${user.name}: ${user.age} років`); }); console.log(""); // Створення копії масиву для сортування // (щоб не змінювати оригінальний масив) const sortedUsers = [...users].sort((a, b) => a.age - b.age); console.log("Відсортований масив (за віком, зростання):"); console.log(sortedUsers); console.log(""); console.log("Користувачі після сортування:"); sortedUsers.forEach(user => { console.log(`${user.name}: ${user.age} років`); }); console.log(""); // Додатково: сортування за спаданням const sortedUsersDesc = [...users].sort((a, b) => b.age - a.age); console.log("Бонус - сортування за спаданням:"); console.log(sortedUsersDesc); alert(`Відсортовано за віком (зростання):\n\n${sortedUsers.map(u => `${u.name}: ${u.age}`).join("\n")}\n\nДив. Console для деталей`); }

Пояснення:

  • sort() - сортує масив на місці (змінює оригінальний масив)
  • [...users] - створення копії масиву (spread operator)
  • (a, b) => a.age - b.age - функція порівняння для сортування за зростанням
  • a.age - b.age - якщо результат негативний, a йде перед b
  • b.age - a.age - для сортування за спаданням
  • Функція порівняння повертає: <0 (a перед b), 0 (рівні), >0 (b перед a)
