Завдання 2.7 - Calculator клас (Method Chaining)

Умова: Розробити клас Calculator, який дозволяє виконувати арифметичні операції над числом за допомогою методів класу, підтримуючи ланцюжковий виклик (method chaining).

Код програми:

class Calculator { constructor() { this.value = 0; } // Встановлює початкове значення number(value) { this.value = value; console.log(`Встановлено значення: ${value}`); return this; // Повертаємо this для ланцюжка } // Додає value до поточного значення add(value) { this.value += value; console.log(`Додано ${value}, результат: ${this.value}`); return this; } // Віднімає value від поточного значення subtract(value) { this.value -= value; console.log(`Віднято ${value}, результат: ${this.value}`); return this; } // Множить поточне значення на value multiply(value) { this.value *= value; console.log(`Помножено на ${value}, результат: ${this.value}`); return this; } // Ділить поточне значення на value divide(value) { if (value === 0) { throw new Error("Ділення на нуль неможливе!"); } this.value /= value; console.log(`Поділено на ${value}, результат: ${this.value}`); return this; } // Повертає поточний результат getResult() { return this.value; } } function testCalculator() { console.log("=== ТЕСТУВАННЯ CALCULATOR ==="); console.log(""); // Тест 1: Базовий приклад з завдання console.log("Тест 1: (10 + 5 - 3) * 4 / 2"); const calc1 = new Calculator(); const result1 = calc1 .number(10) .add(5) .subtract(3) .multiply(4) .divide(2) .getResult(); console.log(`Результат: ${result1}`); console.log(""); // Тест 2: Складніші обчислення console.log("Тест 2: (100 - 50) / 5 + 20"); const calc2 = new Calculator(); const result2 = calc2 .number(100) .subtract(50) .divide(5) .add(20) .getResult(); console.log(`Результат: ${result2}`); console.log(""); // Тест 3: Ділення на нуль (помилка) console.log("Тест 3: Спроба ділення на нуль"); try { const calc3 = new Calculator(); calc3.number(10).divide(0); } catch (error) { console.log(`Помилка: ${error.message}`); } console.log(""); alert(`Тестування завершено!\n\nРезультат 1: ${result1}\nРезультат 2: ${result2}\n\nДив. Console для деталей`); }

Пояснення:

  • class Calculator - оголошення класу
  • constructor() - ініціалізація початкового значення
  • return this - повернення самого об'єкта для ланцюжкових викликів
  • number(value) - встановлює початкове значення
  • add/subtract/multiply/divide - арифметичні операції
  • getResult() - повертає фінальний результат
  • Method chaining - можливість викликати методи один за одним
  • throw new Error() - викидання помилки при діленні на нуль
