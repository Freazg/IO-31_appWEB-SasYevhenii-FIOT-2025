Завдання 1.8 - Storage (Функція-конструктор)

Умова: Написати функцію конструктор Storage який створює об'єкти для управління складом товарів. При виклику отримуватиме один аргумент - початковий масив товарів, і записувати їх у властивість items.

Код програми:

// Функція-конструктор Storage function Storage(initialItems) { this.items = initialItems; // Метод getItems - повертає масив товарів this.getItems = function() { return this.items; }; // Метод addItem - додає новий товар this.addItem = function(item) { this.items.push(item); console.log(`Товар "${item}" додано до складу`); }; // Метод removeItem - видаляє товар якщо він є this.removeItem = function(item) { const index = this.items.indexOf(item); if (index !== -1) { this.items.splice(index, 1); console.log(`Товар "${item}" видалено зі складу`); } else { console.log(`Товар "${item}" не знайдено на складі`); } }; } // Тестування Storage function testStorage() { console.log("=== ТЕСТУВАННЯ STORAGE ==="); console.log(""); // Створення складу з початковими товарами const arr = ["apple", "banana", "mango"]; const storage = new Storage(arr); console.log("Початковий склад:"); console.log(storage.getItems()); console.log(""); // Додавання товарів console.log("Додаємо товари:"); storage.addItem("orange"); storage.addItem("grape"); console.log("Склад після додавання:"); console.log(storage.getItems()); console.log(""); // Видалення товарів console.log("Видаляємо товари:"); storage.removeItem("banana"); storage.removeItem("kiwi"); // Товару немає на складі console.log("Склад після видалення:"); console.log(storage.getItems()); console.log(""); alert(`Поточний склад містить ${storage.getItems().length} товарів\n\nДив. Console для деталей`); }

Пояснення:

  • function Storage() - функція-конструктор (викликається з new)
  • this.items - властивість об'єкта для зберігання товарів
  • this.getItems - метод для отримання масиву товарів
  • this.addItem - метод для додавання нового товару
  • this.removeItem - метод для видалення товару (якщо він є)
  • indexOf() - пошук індексу товару в масиві
  • splice() - видалення елемента з масиву
