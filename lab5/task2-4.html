Завдання 2.4 - Квадрати чисел (map)

Умова: Дано масив чисел [1, 2, 3, 4, 5]. Створіть новий масив, що містить квадрати кожного елементу вхідного масиву. Очікуваний результат: [1, 4, 9, 16, 25].

Код програми:

const numbers = [1, 2, 3, 4, 5]; function calculateSquares() { console.log("=== ОБЧИСЛЕННЯ КВАДРАТІВ ==="); console.log("Початковий масив:"); console.log(numbers); console.log(""); // Створення нового масиву з квадратами чисел const squares = numbers.map(num => num * num); console.log("Масив квадратів:"); console.log(squares); console.log(""); // Виведення деталей обчислень console.log("Деталі обчислень:"); numbers.forEach((num, index) => { console.log(`${num}² = ${squares[index]}`); }); // Альтернативний запис з Math.pow const squaresAlt = numbers.map(num => Math.pow(num, 2)); console.log(""); console.log("Альтернативний спосіб (Math.pow):"); console.log(squaresAlt); // Альтернативний запис з ** (оператор піднесення до степеня) const squaresAlt2 = numbers.map(num => num ** 2); console.log(""); console.log("Альтернативний спосіб (оператор **):"); console.log(squaresAlt2); alert(`Початковий масив: [${numbers.join(", ")}]\n\nКвадрати: [${squares.join(", ")}]\n\nДив. Console для деталей`); }

Пояснення:

  • map() - створює новий масив з результатами виклику функції для кожного елемента
  • num => num * num - стрілочна функція для обчислення квадрата
  • num * num - множення числа на саме себе
  • Math.pow(num, 2) - альтернативний спосіб піднесення до квадрата
  • num ** 2 - оператор піднесення до степеня (ES2016)
  • Не змінює початковий масив, створює новий
